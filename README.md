# unicheck

`unicheck` is a small desktop app for organizing university courses and tasks. It is built in Go with the Fyne GUI toolkit and stores its data locally in your user config directory.

## Build

```bash
go build -o unicheck ./cmd/unicheck
```

## Run

Run the built binary:

```bash
./unicheck
```

Or run it directly with Go:

```bash
go run ./cmd/unicheck
```
